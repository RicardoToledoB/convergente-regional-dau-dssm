# Mejoras v3

## Frontend Angular Material

- Layout institucional con sidebar y tarjetas KPI.
- Tablas Angular Material.
- Paginación servidor para atenciones, eventos, errores y usuarios.
- Filtros por DAU, RUN, paciente, motivo, estado, establecimiento, categoría y fechas.
- Detalle DAU con pestañas: resumen, paciente, admisión, categorización, atención clínica, alta y eventos.
- Visor de JSON original con opción copiar.
- Módulo de creación de usuarios.

## Backend

- Endpoints paginados y filtrados.
- Búsqueda avanzada en atenciones y eventos.
- Endpoint de detalle de evento.
- Endpoints de administración de usuarios.
- Campos adicionales de usuario: email, proveedor/origen, fechas de auditoría y último acceso.
